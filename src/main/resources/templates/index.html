이 사이트는 제가 다룰 수 있는 백엔드 기술들을 소개하는 포트폴리오입니다.

CRUD 데모

JPA를 활용한 CRUD 작업을 보여주는 데모입니다. 사용자 인터페이스를 통해 데이터의 생성, 읽기, 업데이트, 삭제를 직접 체험해보세요.

Redis 데모

Redis를 사용하여 데이터 캐싱과 세션 관리를 어떻게 효율적으로 수행할 수 있는지 보여줍니다. 실시간 데이터 처리의 속도와 유연성을 경험해보세요.

WebSocket 데모

웹소켓을 이용한 실시간 통신 기능을 실습해볼 수 있는 데모입니다. 사용자간 실시간 채팅이나 데이터 스트리밍을 체험해보세요.

인증 데모

JWT와 스프링 시큐리티를 활용한 인증 및 권한 부여 프로세스를 보여줍니다. 로그인, 회원가입, 접근 제어 등의 보안 기능을 확인해보세요.

배치 데모

스프링 배치를 이용한 대량 데이터 처리 및 배치 작업의 자동화를 보여주는 데모입니다. 효율적인 데이터 처리 방법과 스케줄링 기능을 살펴보세요.

API 문서 데모

Swagger 또는 Spring Rest Docs를 사용하여 API 문서를 자동으로 생성하는 방법을 보여줍니다. API의 명세를 쉽게 확인하고 테스트할 수 있습니다.
